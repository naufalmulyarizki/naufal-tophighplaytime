# Naufal Top Higher Playtime

Script peringkat tertinggi playtime / waktu lama bermain di server (REDM)

# Features 
- Optimize, data dibuat ketika loaded player dan tersimpan ke database ketika keluar server
- Bisa diatur berapa banyak peringkat tertinggi playtime yang terlihat
- license bisa di atur, bisa menggunakan license rockstar atau steam

# Installation
- Download Resource
- Masukkan ke dalam resource kamu
- refresh dan start script
